# 📊 Análise do Mercado de Capitalização Brasileiro

Análise exploratória do mercado de capitalização brasileiro utilizando dados públicos disponibilizados pela SUSEP (Superintendência de Seguros Privados).

## 🎯 Objetivo

Explorar o comportamento do mercado de capitalização brasileiro entre 2019 e 2024, identificando tendências, players dominantes e oportunidades de mercado.

## ❓ Perguntas de Negócio

- Quais empresas dominam o mercado e como esse cenário evoluiu nos últimos anos?
- Quais modalidades estão crescendo ou perdendo espaço?
- Como a arrecadação se comportou em momentos de crise, como em 2020?
- Existe concentração regional no mercado?

## 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- Plotly
- Jupyter Notebook

## 📁 Estrutura do Projeto

- `data/raw` — dados brutos extraídos da SUSEP
- `notebooks` — análises e visualizações
- `src` — scripts de tratamento de dados
- `images` — gráficos exportados

## 📌 Fonte dos Dados

Superintendência de Seguros Privados (SUSEP)
https://www.susep.gov.br
